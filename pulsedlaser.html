Key Benefits


FLEXIBLE

넓은 파장대의 레이저를 활용하여, GaN이나 SiC등의 Wide Bandgap 반도체부터 기존 실리콘 소자까지 폭넓게 평가를 지원합니다.

FAST

직관적인 GUI 제어와 자동 스캔 기능을 가진 QRT-LITE 소프트웨어를 통해 테스트 효율성과 분석 품질을 동시에 향상시킵니다.

COST-EFFECTIVE

QRT의 레이저 기반 평가는 방사선 가속기 시험의 높은 비용과 긴 대기 시간을 없애, 보다 빠르고 유연한 평가를 제공합니다. 이를 통해 시험 1회당 비용을 획기적으로 절감하고 연구 효율성을 극대화합니다.

PRECISE ANALYSIS

QRT의 레이저 기반 SEE(Single Event Effect) 평가는 펨토초 레이저 펄스를 활용해 반도체 소자에 미치는 우주 방사선의 영향을 정밀하게 모사합니다.

EXPERIMENT

40년 이상 축적된 반도체 신뢰성 평가 및 불량 분석 경험을 바탕으로, 고객 요구에 부합하는 최고 수준의 레이저 테스트를 구현합니다.





Laser Services



Error Rate Analysis

SEE (Single Event Effect) 평가는 디바이스의 에러 발생 민감도와 내방사선 성능을 정량적으로 확인할 수 있는 핵심 시험입니다. 레이저 조사를 통해 발생하는 Single Event Upset (SEU)현상을 관측하거나 SEL Screening을 통하여 Cross-Section (단면적) 값을 도출합니다. 이 Cross-Section은 LETL (Linear Energy Transfer Threshold) 산출에 사용되며, 이를 통해 실제 우주 환경 또는 방사선 환경에서의 오류율(Error Rate)을 예측하여 제공합니다.
Cross-section (σ)

Sensitive Area Analysis

큐알티는 나노미터 단위로 정밀 제어가 가능한 고해상도 XYZ 스테이지와 레이저 스캐닝 기술을 활용하여, 반도체 칩 상의 특정 위치에 레이저를 정밀 조사합니다. 이를 통해 SEE 발생 위치를 정확히 파악하고, 회로 내 민감 영역(Sensitive Area)을 가시화하여 제공합니다

QRT의 첨단 레이저 평가 시설을 필요할 때 편리하게 대여하실 수 있습니다. 직관적이고 사용자 친화적인 QRT-LITE 소프트웨어를 통해, 복잡한 설정 없이 쉽고 빠르게 SEE 테스트를 진행할 수 있습니다. 레이저 시스템의 작동 원리와 세부 기능은 QRT-LITE Manual에서 확인 가능하며, 처음 사용하는 분들도 신속하게 익혀 효율적인 실험을 수행할 수 있습니다.

QRT Laser Evaluation System



SEREEL2

SEREEL2는 고정밀 레이저 기반 SEE(Single Event Effects) 테스트 솔루션으로, 우주·항공용 반도체의 신뢰성 평가에서 가속기 테스트의 한계를 극복합니다. QRT에서는 630nm에서 1700nm까지의 넓은 파장대를 지원하여 SPA 및 TPA 방식으로 레이저 조사가 가능하며, 자동 스캔, 오토포커스, Chip 이미지와 Stage의 동기화 등의 기능으로 미세한 민감 영역까지 쉽고 빠르게 분석할 수 있습니다. 산업용 레이저의 안정성과 다축 스테이지의 고정밀 이동 기술이 결합되어 R&D부터 대량 스크리닝까지 유연한 대응이 가능합니다.

QRT-LITE (Laser Irradiation Tools for Evaluation)

QRT-LITE는 QRT가 독자 개발한 레이저 기반 SEE(Single Event Effect) 평가 지원 소프트웨어입니다. 펨토초 레이저 시험 환경에서 설정 → 구동 → 수집 → 분석 전 과정을 하나의 화면에서 직관적으로 제어할 수 있는 통합 플랫폼으로, 레이저 파장, 에너지 등의 조건을 저장하고 불러오는 기능을 통해 평가의 재현성과 일관성을 확보하였고, 오실로스코프와 소스미터 등 계측기를 연동해 전기적 응답을 동기 수집합니다. 또한 칩 이미지와 좌표를 매핑해 원하는 지점으로 스테이지를 정밀하게 이동할 수 있으며, 스캔 예약 기능을 통해 무인 자동화 운용이 가능합니다. 이를 통해 우주·항공용 반도체의 신뢰성 검증을 빠르고 정밀하게 지원합니다.
QRT-LITE Manual에서 설치부터 장비 연동, 맵핑/스캔 설정, 자동화 기능, Set-up까지 모든 기능을 단계별로 확인할 수 있습니다.
