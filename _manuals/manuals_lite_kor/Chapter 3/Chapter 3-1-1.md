# 3.1.1. 간단한 조작버튼

-	Open: 이미지 파일 열기
-	Save: 저장
-	Zoom in: 이미지 확대
-	Zoom out: 이미지 축소
-	Rotate: 이미지 회전
-	Sample Image: Chip 이미지 없을 경우 Sample 이미지 (Figure 10의 격자 그림)
-	Set Reference Point: 현재 위치에 Reference Point 생성 (핑크색)
-	Remove Reference Point: Reference Point 삭제
