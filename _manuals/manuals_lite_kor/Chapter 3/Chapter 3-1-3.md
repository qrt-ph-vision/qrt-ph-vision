# 3.1.3. 이미지 좌표 입력

수동으로 Point0~3까지 좌표를 입력할 수 있는 영역입니다. Chip Image에서 마우스 우 클릭 후 'Put Current pos(position) to p0~3'를 통해 좌표를 지정하고 Apply시 자동으로 좌표가 입력됩니다.
